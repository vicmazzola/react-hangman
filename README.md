## Installation
To run this project locally:

1. Clone the repository and navigate to the project directory:
   ```
   git clone <https://github.com/vicmazzola/react-hangman>
   cd react-hangman
   ```

2. Install dependencies:
```
npm install
```
3. Start the development server:
```
npm run dev
```
